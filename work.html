Meet App

A Progressive Web App with offline capabilities for viewing meet-ups pulled from the Google Calendar API. The App allows users to filter by city and number of events and also displays a chart of meetings by subject and a graph of meetings by city. Meet App was created with React and has a 100% testing coverage with test-driven development technique (TDD). For the authorization server it uses AWS Lambda.

Chat App with React Native

Native Mobile Chat App created with React Native and Expo. The Chat App uses Firebase for the database and the Gifted Chat library for building the chat interface. The features include adding your name and selecting a different background color that carries over to the chat view. You can use the React Native elements to use the phone camera, GPS location to share a picture or your location.
